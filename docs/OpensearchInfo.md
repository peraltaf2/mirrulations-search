## Database Overview

There are three OpenSearch indices:

- `comments`
- `comments_extracted_text`
- `documents`
- `extracted_text_test` (testing only)

---

## `comments` Index

```json
{
  "comments": {
    "mappings": {
      "properties": {
        "commentId": {
          "type": "text",
          "fields": {
            "keyword": {
              "type": "keyword",
              "ignore_above": 256
            }
          }
        },
        "commentText": {
          "type": "text",
          "fields": {
            "keyword": {
              "type": "keyword",
              "ignore_above": 256
            }
          }
        },
        "docketId": {
          "type": "text",
          "fields": {
            "keyword": {
              "type": "keyword",
              "ignore_above": 256
            }
          }

}
```
Has around 25 million json files
